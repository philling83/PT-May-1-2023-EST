Selection and Manipulation Practice Project

In this project, you will start with a collection of eight squares. In the your-code.js file, you will see a list of tasks. Each task will ask you to select an element or elements and manipulate them in some way. Each time you refresh the page, you should see your manipulations rendered on the screen.

You may only add JavaScript code to the your-code.js file. Do not change any code on the site.css or index.html files.
